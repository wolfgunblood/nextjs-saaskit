## Get Started

1. `npx nextjs-saaskit`
2. `cd projectName`
3. Copy `.env.loc` to `.env.local`
4. Copy `.env.example` to `.env`
5. Update env variables
6. Run `pnpm install` to install dependencies.
7. Execute `npm dev` to start the development server and enjoy!
